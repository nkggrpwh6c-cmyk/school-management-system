File Format Guide
Supported Formats:
  • CSV (.csv)
Required Columns:
  • first_name - Student's first name
  • last_name - Student's last name
  • email - Student's email address
  • lrn - Learner Reference Number
  • grade_level - Grade level (1-12)
  • section - Section/Class name
Optional Columns:
  • middle_name - Middle name
  • date_of_birth - Birth date
  • gender - Male/Female
  • address - Home address
  • phone_number - Contact number
  • parent_name - Parent/Guardian name
  • parent_phone - Parent contact
  • parent_email - Parent email
  • emergency_contact - Emergency contact
  • blood_type - Blood type
  • medical_conditions - Medical info
  • allergies - Known allergies

Pro Tips
  • Use the template to ensure correct column names
  • Make sure all required fields are filled
  • LRN must be unique for each student
  • Email addresses must be valid and unique
  • Grade level should be a number between 1-12
